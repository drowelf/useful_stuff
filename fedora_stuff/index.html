Robust Install of VirutalBox on Fedora 42

Fedora + VirtualBox can be a little inconsistent..Fedoraâ€™s kernel updates more quickly and VirtualBox. So do this:

  1. Make sure we have access to the RPMFusion package
sudo dnf install https://mirrors.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm
  1. Install VirtualBox and kernel modules
sudo dnf install VirtualBox
  1. Install build dependencies
sudo dnf install gcc make perl kernel-devel kernel-headers dkms
  1. Add yourself to the vboxusers group
sudo usermod -aG vboxusers $USER
  1. Log out/in (or reboot) to apply.
sudo akmods
sudo modprobe vboxdrv
# Optional
# reboot now
  1. Keeping things working after Fedora kernel updates

Confirm modules are rebuilt correctly, run after a kernel update:

bash systemctl status akmods
